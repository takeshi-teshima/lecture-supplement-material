## サポートページ

## ローカル実行のためのセットアップ
```
$ pip install -U pip
$ pip install poetry
$ poetry install
```

## ローカル実行
1. VSCodeで各ノートブックを開いて，`.venv`をカーネルとして選択して実行する．
1. もしくは，`$ poetry run jupyter lab` を実行してノートブックを実行する．
